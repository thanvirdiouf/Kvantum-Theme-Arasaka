# Kvantum-Theme-Arasaka
Kvantum theme inspired by the red Arasaka terminals from the game Cyberpunk 2077

---

![Screenshot of Dolphin](https://raw.githubusercontent.com/thanvirdiouf/Kvantum-Theme-Arasaka/refs/heads/main/Arasaka.png)
## Installation
1.  Clone this repo into ~/.config/Kvantum
```
git clone --depth 1 https://github.com/thanvirdiouf/Kvantum-Theme-Arasaka/ ~/.config/Kvantum/Arasaka
```
2.   Open kvantum manager(install using package manager).
3.   Click on `Change/Delete Theme` and select `Arasaka`
